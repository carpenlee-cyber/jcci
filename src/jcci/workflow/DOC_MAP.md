## 💡 使用技巧

### 搜索文档内容

在VS Code中：
```
Ctrl+Shift+F → 输入关键词 → 选择workflow目录
```

### 快速跳转

使用Markdown链接：
```markdown
[配置说明](README_STREAMLIT.md#配置步骤)
```

### 打印文档

```bash
# 转换为PDF（需要pandoc）
pandoc QUICKSTART.md -o quickstart.pdf
```
